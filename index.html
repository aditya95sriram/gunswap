Gunswap is a web-based juggling animator. Patterns are defined using standard siteswap notation with support for synchronous, multiplex, bouncing and passing. Some examples to get you started:

  • 5 - 5 ball cascade
  • 7531 - 4 ball tower
  • 91 - 5 ball shower
  • (4,4) - 4 ball sync fountain
  • (4,2x)(2x,4) - 3 ball box
  • 24[54] - 5 ball multiplex
  • 5B1L - 5 ball lift bounce
  • 5S05S3 - 5 club alternating flats/triples
  • <3|3><3P|3P> - 6 ball 2 juggler passing
  • <5|5><5PB1F|5PB1F> - 6 ball 2 juggler passing w/ force bounce
  • <7|5|3> - 3 jugglers no passing

Check out http://gunswap.blogspot.com/ for updates on development.

Gunswap uses some really awesome open source projects:

  • Three.js - for 3D graphics
  • Bootstrap - to make UI controls pretty
  • jQuery - for anything and everything
  • Grunt - to aid development/testing
  • Mocha - for testing
  • Travis CI - for continuous integration
  • GitHub - for hosting
